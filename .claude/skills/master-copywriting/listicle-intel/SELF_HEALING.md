Capture VALUE LESSONS specific to listicle copy and offer analysis. Lessons here can propagate to:
- Master-copywriting skill (if universal)
- Other offer types (if pattern applies)

## Review Checklist (Listicle-Specific)

Before finalizing any listicle copy:

- [ ] Checked intel exists for this offer type
- [ ] Intel is < 30 days old (or refreshed)
- [ ] Used learned CTA patterns (not generic)
- [ ] Applied offer-specific psychology
- [ ] Competitive advantages documented
- [ ] Structure matches winning patterns

---

## Approval Criteria

### Auto-Approve When:
- Using patterns from offer type with 3+ successful uses
- Following documented best practices
- CTAs match proven winners

### Request Review When:
- New offer type (no prior intel)
- Deviating from established patterns
- Testing new approach

## Cross-Skill Propagation Rules

### Propagate to Master-Copywriting When:
- Lesson applies to ALL copy types (not just listicles)
- CTA patterns work universally
- Psychology triggers are general

### Propagate to Specific Offer Type When:
- Lesson is category-specific
- Pattern only works for that offer type
- Competitor intel is offer-specific

### Keep in Listicle-Intel Only When:
- Research methodology issue
- Brightdata-specific problem
- Storage/organization issue
