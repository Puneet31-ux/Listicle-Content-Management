## How to Add a Lesson

When capturing feedback:

1. Identify the specific issue
2. Extract the learning
3. Categorize appropriately
4. Add entry to relevant section above
5. Update the Index table
6. If universal → propagate to master-copywriting
7. If offer-specific → add to that offer type's files
