# Listicle Headline Patterns

## Purpose

Proven headline patterns extracted from competitor analysis. Use these as starting points, not templates to copy verbatim.

---

## Pattern Categories

### 1. Number + Benefit Headlines
```
[X] Ways to [Achieve Benefit] in [Timeframe]
[X] [Category] Options That Actually [Deliver Result]
Top [X] [Category] for [Audience] in [Year]
```

**Best For:** Comparison listicles, resource guides
**Psychology:** Specificity builds trust, numbers promise structure

---

### 2. Question Headlines
```
Looking for [Solution]? Here's What You Need to Know
Which [Category] Is Right for You? [Year] Guide
Still [Pain Point]? These [X] [Solutions] Can Help
```

**Best For:** Problem-aware audiences, search intent matching
**Psychology:** Mirrors internal dialogue, creates resonance

---

### 3. Comparison Headlines
```
[Option A] vs [Option B]: Which [Result] in [Year]?
The [X] Best [Category] Compared (We Tested Them All)
[Category] Showdown: [X] Options Ranked by [Criteria]
```

**Best For:** Decision-stage readers, competitive analysis
**Psychology:** Reduces cognitive load, positions as helpful

---

### 4. Urgency/Timeliness Headlines
```
[Year] Update: [X] [Category] Worth Considering Now
Before You [Action], Read This [Category] Comparison
[X] [Category] You Should Know About Before [Event/Deadline]
```

**Best For:** Time-sensitive offers, seasonal content
**Psychology:** Creates FOMO, implies current relevance

---

### 5. Authority Headlines
```
Expert-Reviewed: [X] [Category] That Deliver
[X] [Category] Recommended by [Authority Type]
The [Category] [Professionals] Actually Use
```

**Best For:** High-stakes decisions, trust-building
**Psychology:** Borrowed credibility, reduces risk perception

## Offer-Type Specific Patterns

*(Populated from Brightdata research per offer type)*

### Financial (Debt, Loans, Credit)
```
[X] Ways to [Reduce/Consolidate/Improve] Your [Financial Issue]
[Year]'s Top [X] [Financial Product] for [Situation]
Compare [X] [Financial Service] Options (Rates from [X]%)
```

### Home Services (Solar, Roofing, HVAC)
```
[X] [Service] Companies in [Area] (Free Quotes Available)
How Much Does [Service] Really Cost? [X] Options Compared
The [X] Best [Service] Companies for [Home Type/Situation]
```

### Health & Wellness
```
[X] [Treatment/Product] Options for [Condition]
Compare [X] [Health Service] Providers Near You
[X] [Health Product] That Actually [Work/Deliver]
```

## Testing Recommendations

### A/B Test These Elements:
- Number (5 vs 7 vs 10)
- Year inclusion (2025 vs omitted)
- Question vs statement format
- Specificity level (general vs targeted)

### Track:
- CTR from search results
- Time on page
- Scroll depth
- Conversion rate
