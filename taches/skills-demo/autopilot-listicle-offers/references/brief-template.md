## Usage Notes

### When to Use This Template
- At Checkpoint #3 (after all research complete)
- Before auto-invoking Layer 1 & 2
- As comprehensive handoff document

### Required Sections
- Offer Overview
- Target Audience
- Research Findings (all 4 tracks)
- Preliminary Answers to Layer 1 Questions
- Confidence Assessment
- Compliance Alerts

### Optional Sections
- Can skip Secondary Persona if not applicable
- Can skip Urgency if no authentic urgency exists
- Can skip certain research tracks if category doesn't require (e.g., healthcare compliance for non-health offers)

### Confidence Levels
- **HIGH:** Validated across 3+ sources, consistent data
- **MEDIUM:** Some data, logical inferences, needs validation
- **LOW:** Gaps, contradictions, or pure speculation

Always flag confidence levels explicitly.
